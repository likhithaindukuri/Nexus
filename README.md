# KnowledgeBot AI

A Multi-Tenant AI Knowledge Platform built using React, FastAPI, PostgreSQL, ChromaDB, and RAG.

## Features

- Multi Tenant Support
- Authentication
- PDF Upload
- AI Chatbot
- RAG Pipeline
- Widget Generator
- Analytics Dashboard
- Citation Support

## Tech Stack

Frontend: React + Tailwind

Backend: FastAPI

Database: PostgreSQL

Vector Database: ChromaDB

AI: Gemini API
